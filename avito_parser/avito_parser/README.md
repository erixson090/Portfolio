# 🕷️ Парсер Avito с обходом антибот защиты

Парсер для сбора объявлений с Avito с обходом блокировок через антидетект браузер AdsPower и резидентные прокси 9Proxy.

---

## 📊 Результаты работы

| Показатель | Значение |
|------------|----------|
| **Загружено данных** | 184 289 символов |
| **Время загрузки** | ~11 секунд |
| **Найдено товаров** | 36+ с ценами |
| **Статус блокировки** | HTTP 403 → 200 OK |

---

## 🛠 Технологический стек

| Компонент | Назначение |
|-----------|------------|
| **Python 3.12** | Основной язык |
| **AdsPower** | Антидетект браузер |
| **9Proxy** | Резидентные прокси |
| **Crawl4AI** | Загрузка страниц |
| **Ollama (llama3.2)** | Локальный ИИ |

---

## 🏗 Архитектура

AdsPower → 9Proxy → Crawl4AI → Ollama → JSON

---

## 🚀 Установка и запуск

### 1. Клонировать репозиторий

```bash
git clone https://github.com/erixson090/avito-parser.git
cd avito-parser

```

### 2. Установить Ollama и скачать модель

ollama pull llama3.2:3b

### 3. Настроить переменные окружения

Создай файл .env:

AI_API_KEY=sk-твой_ключ
ADS_PROFILE_ID=твой_id_профиля

###4. Запустить парсер

python parser.py

  ## 🧠 Как обходятся блокировки

| Проблема | Решение |
|----------|---------|
| 🔴 HTTP 403 Forbidden | 🌐 **Резидентные прокси 9Proxy** — IP реальных пользователей из жилых сетей |
| 🤖 Детект автоматизации | 🛡️ **Антидетект AdsPower** — подмена WebGL, Canvas, WebRTC, User-Agent, разрешения экрана |
| ⏳ Динамическая загрузка контента | 📜 **Автоматическая прокрутка страницы** через Crawl4AI (эмуляция скролла) |
| 🧩 Капчи и поведенческие факторы | 👤 **Полноценный браузер с эмуляцией человека** — случайные задержки, движения мыши |
